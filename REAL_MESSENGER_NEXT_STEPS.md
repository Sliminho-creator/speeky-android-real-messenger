# Speeky real messenger next steps

## Что уже есть в этом build
- локальная регистрация и вход
- локальные чаты и голосовые
- экран звонка
- редактирование профиля: ник, username, описание, фото
- настройка масштаба интерфейса

## Следующий реальный этап
1. Backend API (Node.js/NestJS или Go)
2. PostgreSQL для аккаунтов, чатов, сообщений
3. WebSocket для онлайна и мгновенной доставки
4. S3-совместимое хранилище для фото, файлов и голосовых
5. WebRTC signalling server для звонков и кружочков
6. Firebase / push-уведомления
7. JWT + refresh tokens + защита сессий
