# Speeky Android Real Messenger v2

Обновлённый Android-проект в стиле твоего `base.apk`.

Что я поправил в этой версии:
- добавил **регистрацию + вход** в одном auth-экране (переключение сверху);
- убрал раздражающий мгновенный звук при нажатии на звонок — теперь открывается экран звонка;
- оставил **одну** основную кнопку записи/отправки в чате;
- добавил **launcher icon / round icon / adaptive icon**;
- сделал отдельный **логотип приложения** в папке `branding/`;
- оставил все основные экраны кликабельными: чаты, детали контакта, звонок, видео, профиль, устройства, оформление, настройки.

## Что реально работает сейчас
Это **живой Android UI-проект**, который собирается в APK и даёт рабочие пользовательские сценарии на устройстве:
- локальная регистрация;
- локальный вход;
- список чатов;
- открытие диалога;
- отправка текстовых сообщений в локальное состояние;
- экран звонка / видео-звонка;
- переходы по настройкам;
- смена темы внутри приложения.

## Что ещё не является полноценным серверным мессенджером
Сейчас тут **нет готового production-backend** для:
- реального общения между разными устройствами;
- WebRTC-звонков между людьми;
- push-уведомлений через FCM;
- синхронизации истории между аккаунтами.

Если захочешь, следующим шагом можно уже строить **серверную часть**: auth + websocket + WebRTC signaling + push.

## GitHub Actions сборка APK
Проект содержит workflow:
`.github/workflows/android-apk.yml`

После пуша в GitHub:
1. Открой **Actions**
2. Запусти **Build Android APK**
3. После успешной сборки скачай артефакт `speeky-debug-apk`

## Локальная структура
- `android/` — Android проект
- `branding/` — логотип и иконки
